# 画面仕様

## 1. Room画面（視聴者）

**パス**: `/room`
**用途**: 視聴者が作業セッションに参加・応援する画面

### 1.1 レイアウト

```
┌─────────────────────────────────────┐
│ ← 戻る  [配信者名]           [Theme]│ ← Header
├─────────────────────────────────────┤
│ [作業中]  参加者: 12人      45:23   │ ← Status Bar
├─────────────────────────────────────┤
│                                     │
│  今回の宣言                         │
│  デザインシステムの設計と...        │ ← Declaration
│                                     │
├─────────────────────────────────────┤
│           [ 完了！ ]                │ ← Complete Button
├─────────────────────────────────────┤
│  応援する（交換性なし・演出のみ）   │
│  [¥300] [¥500] [¥1000] [¥3000]     │ ← Support Buttons
│  ※応援は配信者への支援金です...     │
├─────────────────────────────────────┤
│  チャットは休憩中のみ利用可能です   │ ← Chat Area
│  （または休憩中のチャットUI）       │
└─────────────────────────────────────┘
```

### 1.2 機能

- **状態表示**: 作業中/休憩中の表示
- **タイマー**: セッション経過時間
- **宣言表示**: 配信者の今回の宣言
- **完了ボタン**: 自分の作業完了を報告
- **応援ボタン**: 300/500/1000/3000円から選択
- **チャット**: 休憩中のみ表示・入力可能

### 1.3 制約

- 応援ボタンには「交換性なし・演出のみ」の注記を表示
- 作業中はチャットエリアを非アクティブ化
- 初見は一定時間発言不可（将来的実装）

## 2. Creator Dashboard（配信者）

**パス**: `/dashboard`
**用途**: 配信者がセッションを管理・分析する画面

### 2.1 レイアウト

```
┌─────────────────────────────────────┐
│ ← 戻る  Dashboard（配信者）  [Theme]│ ← Header
├─────────────────────────────────────┤
│ 参加者: 12人  完了率: 67%  45:23   │ ← Stats Row
├─────────────────────────────────────┤
│ 現在の宣言                          │
│ デザインシステムの設計と...         │ ← Declaration
├─────────────────────────────────────┤
│ 状態切り替え: [作業中] [休憩中]     │ ← Status Control
├─────────────────────────────────────┤
│ 応援者一覧                          │
│ ├─ ユーザーA  ¥1,000  14:30        │
│ ├─ ユーザーB  ¥500    14:25        │
│ └─ ユーザーC  ¥300    14:20        │ ← Supporters List
├─────────────────────────────────────┤
│ 通報キュー                          │
│ 現在通報はありません                │ ← Reports Queue
└─────────────────────────────────────┘
```

### 2.2 機能

- **統計表示**: 参加者数、完了率、経過時間
- **宣言表示・編集**: 現在の宣言を表示（編集は将来的）
- **状態切り替え**: 作業中/休憩中の切り替えボタン
- **応援者一覧**: 応援者名、金額、時間の表示
- **通報キュー**: 通報があった場合の対応リスト

### 2.3 制約

- 配信者のみアクセス可能（将来的な認証実装）
- 通報は即座に表示され、対応が必要

## 3. OBS Overlay

**パス**: `/overlay`
**用途**: OBS等の配信ソフトで使用するオーバーレイ表示

### 3.1 レイアウト

```
┌──────────────────────────────┐
│ 参加者: 12    完了: 67%      │ ← Stats Bar (背景透過)
├──────────────────────────────┤
│ 最新の応援                    │
│ ユーザーAさんが¥1,000で応援！│ ← Latest Support
└──────────────────────────────┘
```

### 3.2 機能

- **統計表示**: 参加者数、完了率
- **最新応援**: 最新の応援者と金額を表示
- **背景透過**: OBSでの合成用に透明背景

### 3.3 制約

- 背景は半透明の黒（rgba(0,0,0,0.7)）
- テキストは白色で可視性確保
- アニメーションは控えめに

## 4. ホーム画面

**パス**: `/`
**用途**: サービス紹介・画面ナビゲーション

### 4.1 レイアウト

```
┌─────────────────────────────────────┐
│ WakuWork Circle              [Theme]│ ← Header
├─────────────────────────────────────┤
│ 画面一覧                            │
│ ┌──────────┐ ┌──────────┐ ┌───────┐ │
│ │ Room     │ │ Dashboard│ │Overlay│ │ ← Cards
│ │ (視聴者) │ │ (配信者) │ │(OBS用)│ │
│ └──────────┘ └──────────┘ └───────┘ │
├─────────────────────────────────────┤
│ コンセプト                          │
│ ・同席作業                          │
│ ・応援（返礼なし）                  │
│ ・交換性禁止                        │
│ ・治安UI強制                        │
└─────────────────────────────────────┘
```

### 4.2 機能

- **画面一覧**: Room/Dashboard/Overlayへのリンク
- **コンセプト説明**: サービスの理念を簡潔に説明
- **テーマ切り替え**: 3テーマの切り替えUI
